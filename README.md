# Band Sessions

Десктоп-приложение для двух (или больше) музыкантов, которые работают над одними и теми же проектами Logic Pro / GarageBand / Auxy через общую папку (Яндекс.Диск).

Решает простую проблему: когда оба участника группы могут открыть один и тот же проект одновременно и случайно перезаписать правки друг друга — без этого приходится вручную договариваться в Telegram, кто сейчас работает.

## Что делает

- Показывает список треков/проектов со статусом: свободно или занято (и кем)
- Блокирует проект на время работы — пока ты не нажал "завершить", партнёр видит, что трек занят
- Автоматически увеличивает номер версии при завершении сессии
- Хранит короткую историю последних изменений (кто, что сделал)

## Как это работает

Приложение не хранит файлы и не лезет внутрь проектов Logic Pro/GarageBand — оно просто кладёт небольшой JSON-файл со статусами внутрь той же папки, которую синхронизирует Яндекс.Диск. Сам Диск синхронизирует этот файл между участниками группы так же, как и сами аудио-проекты — отдельный сервер не нужен.

## Стек

- [Tauri](https://tauri.app/) — десктоп-обёртка (Rust + системный WebView)
- React + TypeScript — интерфейс
- Rust — чтение/запись файла состояния
